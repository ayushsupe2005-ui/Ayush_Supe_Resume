# Ayush_Supe_Resume
This is my resume. I made it in HTML and CSS so I figured that it would be a good idea to share this online on github. 
To print the resume, just open the html document in a web browser and press ctrl + P. 

## The HTML document

### `<head>`
inside the head tags, there are 3 main parts: 
1. Linking the css document: `<link rel="stylesheet" href="style.css">`
2. Importing the font from fonts.google.com (Roboto): 
`<link rel="preconnect" href="https://fonts.googleapis.com">`
`<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>`
`<link href="https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,100..900;1,100..900&display=swap" rel="stylesheet">`
4. A title that is displayed in the browser tab: `<title>resume</title>`

### `<body>`
The body uses the following tags: 
1. `<h1>` tags are used for the name
2. `<h2>` tags are used for labeling the different sections in the resume
3. `<p>` tags are used for the content relating
4. `<ul>` and `<li>` tags are used to create unordered bulleted lists to describe content in the `<p>` tags
5. `<hr>` tags are used to separate the different sections
